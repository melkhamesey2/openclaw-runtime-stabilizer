# Security Policy

## Scope

This repository is defensive and reliability-focused. It documents safe handling of local AI runtime state, session registries, and public redaction practices.

## Do not include in issues

Please do not post:

- private transcripts
- raw chat logs
- tokens or keys
- hostnames
- personal local paths
- confidential third-party data
- unredacted runtime dumps

## Reporting safely

When reporting a similar incident, include only:

- application version
- operating system family
- high-level symptom description
- sanitized counts
- redacted paths
- screenshots with private data removed

## Maintainer policy

Issues or pull requests containing sensitive data may be edited, closed, or removed from public discussion.
