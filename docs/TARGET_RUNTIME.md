# Target Runtime

## OpenClaw version

This public artifact is based on work performed against:

```text
OpenClaw 2026.6.8
```

## Environment class

The incident and stabilization workflow were developed for a local Windows-based OpenClaw installation with local Ollama-backed model execution.

## Compatibility note

The documentation may still help when investigating similar symptoms on nearby OpenClaw versions, but all concrete runtime assumptions in this repository should be treated as version-scoped to OpenClaw `2026.6.8` unless explicitly revalidated.

## Public-edition boundary

This repository is a public-safe documentation and reliability artifact. It does not claim that the same remediation decisions are automatically safe for every OpenClaw version or every local session store.
